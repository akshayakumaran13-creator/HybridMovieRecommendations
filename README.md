🎬 Hybrid Movie Recommendation System

A full-stack intelligent movie recommendation web application that combines Content-Based Filtering and Collaborative Filtering to deliver personalized movie suggestions with enhanced accuracy and user experience.

🚀 Key Features
🔐 User Authentication System
Secure login and registration
Separate user sessions for different users
Each user has isolated recommendation history
🎯 Hybrid Recommendation Engine
Combines:
Content-based filtering (genre, similarity, metadata)
Collaborative filtering (user behavior & ratings)
Provides more accurate and diverse recommendations
⭐ Movie Details Display
Movie name
Genre classification
User ratings / predicted ratings
Movie poster images
Detailed movie description
📊 Personalized Dashboard
Shows recommendations based on individual user history
Stores past interactions and viewed recommendations
Improves suggestions over time
🧠 Intelligent Search & Matching
Handles movie title inputs
Supports fuzzy matching for better user experience
🎨 UI/UX Features
Clean, responsive interface
Movie cards with posters and descriptions
Easy navigation between login, dashboard, and recommendations
🧩 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Flask (Python)
Data Handling: Pandas, NumPy
ML/Recommendation: Content-based + Collaborative filtering
Authentication: Session-based login system
API/Data: TMDB (optional for posters & metadata)
👥 Multi-User System

Each user has:

Separate login credentials
Independent recommendation history
Personalized movie suggestions
No cross-user data leakage
📌 Project Goal

To build a scalable and intelligent movie recommendation system that mimics real-world streaming platforms by combining multiple recommendation strategies and delivering personalized, data-driven movie suggestions.
